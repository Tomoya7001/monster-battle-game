## 1. 基本ルール

### バトル形式
| 項目 | 値 |
|------|-----|
| 対戦形式 | 1対1 |
| 勝利条件 | 相手モンスター3体先に倒す |
| パーティ編成 | 5体から3体選出 |

### コストシステム
| 項目 | 値 |
|------|-----|
| 初期コスト | 4 |
| 最大コスト | 7 |
| 毎ターン回復 | 2 |
| 交代コスト | 0（ただし1ターン無防備） |

### 奥義制限
- バトル中1回のみ使用可能（パーティ全体で1回）
- 奥義コスト: コスト3～6（モンスターごとに異なる）
- カテゴリに関係なく、同コストなら同等の強さ

## 2. 行動順システム

### 基本ルール
```
■ 行動順決定
1. 優先度が高い技を使用した方が先制
2. 同優先度の場合は素早さで判定
3. 同優先度・同素早さの場合はランダム（50%）
```

### 優先度システム
| 優先度 | 分類 | 技例 |
|--------|------|------|
| +3 | 交代時特殊 | 追撃（相手交代時のみ） |
| +2 | 強先制技 | 神速斬、閃光撃 |
| +1 | 通常先制技 | 影縫い、激流撃、疾雷、真空波 |
| 0 | 通常技 | 全ての通常技、防護壁、先読み |
| -1 | 後攻技 | カウンター、魔力返し、報復 |

### 行動順の例
```
例1: 通常技同士（優先度0）
プレイヤーA: 業炎（素早さ60）
プレイヤーB: 天雷（素早さ100）
→ Bが先制（同優先度、素早さ100 > 60）

例2: 先制技 vs 通常技
プレイヤーA: 影縫い（優先度+1、素早さ60）
プレイヤーB: 業炎（優先度0、素早さ100）
→ Aが先制（優先度+1 > 0）

例3: 強先制技 vs 通常先制技
プレイヤーA: 神速斬（優先度+2、素早さ60）
プレイヤーB: 影縫い（優先度+1、素早さ100）
→ Aが先制（優先度+2 > +1）

例4: 後攻技
プレイヤーA: カウンター（優先度-1、素早さ80）
プレイヤーB: 業炎（優先度0、素早さ60）
→ Bが先制（優先度0 > -1）
→ Aはダメージを受けた後にカウンター発動
```

| rear_phy_002 | 報復 | 2 | C | none | 後攻時威力1.5倍（B相当） | クロム(一般種)、タロス(一般種)、クリス(一般種) |

### 後攻技の仕様
```
■ カウンター / 魔力返し
・そのターンにダメージを受けた場合のみ発動
・ダメージを受けなかった場合は不発（コストは消費）
・カウンター: 物理技のダメージのみ対象
・魔力返し: 特殊技のダメージのみ対象

■ 報復
・後攻時: 威力C → B相当（1.5倍）
・先制時: 威力Cのまま
・確実に発動する（不発なし）
```

## 2-3. トリックルーム（時空歪曲） ✅ 実装完了

> **実装先**: skill_masters_common_v6.json (v7.1.0)

### 概要
5ターン間、素早さの判定を逆転させる特殊技。鈍足モンスターを活かす戦術を可能にする。

### 技データ
| 技ID | 技名 | コスト | 威力 | 属性 | 効果 |
|------|------|--------|------|------|------|
| field_time_001 | 時空歪曲 | 4 | - | none | 5T間、同優先度内で素早さが低いほど先に行動 |

### 習得者
| モンスター | カテゴリ | 備考 |
|-----------|---------|------|
| 裏の支配者グラム | 幻種 | メカノイド/雷 |
| 雪女シロネ | 一般種 | スピリット/水 |
| 深淵ゼラ | 一般種 | ミュータント/水 |
| 深海魚シーラ | 一般種 | デーモン/水 |

### 時空歪曲の仕様
```
■ 発動
・優先度: 0（通常技扱い）
・発動ターンは通常の素早さ順で行動
・次のターンから効果開始

■ 効果中の行動順
・優先度は通常通り判定（+2 > +1 > 0 > -1）
・同優先度内のみ素早さ逆転
・例: 素早さ50のモンスターが素早さ100のモンスターより先に行動

■ 効果終了
・5ターン経過で自動解除
・上書き不可（既に時空歪曲中は再使用不可）

■ 戦術例
時空歪曲パーティ（トリパ）:
・グラム（発動役）→ 時空歪曲使用
・タロス（素早さ55）→ 歪曲中は最速級に
・クロム（素早さ50）→ 歪曲中は最速級に
```

## 4. 状態異常システム

### 基本ルール
| 項目 | ルール |
|------|--------|
| 付与確率 | 100%確定（運要素排除） |
| 重ねがけ | 同じ状態異常の重ねがけ不可（持続リセットのみ） |
| 同時付与上限 | 2種類まで |
| 3種類目付与時 | 最も古い状態異常が解除される |

### 状態異常一覧
| 状態異常 | 効果 | 持続 |

| 毒(poison) | 毎T HP2%ダメージ | 3T |

| 瘴気(miasma) | 毎T HP3%ダメージ + 魔力-5% | 3T |
| 麻痺(paralysis) | 素早さ-15% | 3T |
| 拘束(restraint) | 交代不可 | 2T |
| 封印(seal) | 最高コスト技使用不可 | 2T |
| 魅了(charm) | 被ダメージ+10% | 2T |
