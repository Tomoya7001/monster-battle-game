## 6. トリガータイプ定義

### 6.1 基本トリガー

| トリガー | 発動タイミング | 例 |
|----------|---------------|----|
| on_entry | 登場時 | 畏怖の波動、凍てつく美貌 |
| on_hit | 攻撃命中時 | 強欲の力（旧仕様） |
| on_attack_received | 攻撃技を受けた時 | 帯電被膜、反射の鎧 |
| turn_end | ターン終了時 | 再生細胞、聖なる祈り |
| on_kill | 相手を倒した時 | 搾取、魂喰らい |
| on_death | 倒された時 | 最期の呪詛 |
| passive | 常時発動 | 守りの構え、竜鱗 |

### 6.2 複合トリガー

| トリガー | 発動タイミング | 動作 |
|----------|---------------|------|
| **on_entry_and_hit** | 登場時 + 攻撃命中時 | 登場時に1回発動 + 攻撃命中ごとに発動（1T1回制限可） |
| **passive_and_on_hp_cost_skill** | 常時 + HP消費技使用時 | 反骨の魂 |
| **passive_and_on_hit** | 常時 + 攻撃命中時 | 煉獄の炎（v20.0更新） |
| **passive_and_on_cost5_skill** | 常時 + コスト5技使用時 | 深淵の目覚め（v20.0更新） |
| **on_switch_out** | 交代する時 | 置き土産 |
