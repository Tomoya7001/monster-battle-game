## 1. 概要

アビリティはモンスターが持つパッシブ能力。常時発動または特定条件で自動発動し、バトルを有利に進める。

### 1.1 設計思想

| 原則 | 内容 |
|------|------|
| 効果数 | **1つのみ**（猪突猛進のみ例外） |
| HP条件 | **3種類のみ**（満タン/半分以上/半分以下） |
| 倍率表示 | **内部処理**（プレイヤーには見せない） |
| 効果文 | **短く直感的に** |
| 運要素 | **完全排除**（ランダム効果禁止） |

### 1.2 アビリティ枠とプール

| 項目 | 内容 |
|------|------|
| アビリティ枠 | **1つのみ** |
| プール数 | モンスターごとに2〜5個 |
| 選択方法 | 所持プールから装備のように選択 |

### 1.3 アビリティの習得方法

| 方法 | 説明 |
|------|------|
| ガチャ取得時 | プールからランダムで1つ習得、デフォルトでセット |
| 継承 | 同モンスターを素材にしてアビリティを確定で渡す（最大5体まで選択可） |
| アイテム | 課金アイテム、パス景品、ショップ、イベントなどで好きなアビリティを選択可 |

## 2. HP条件（3種類のみ）

| 条件名 | 内部値 | 説明 |
|--------|--------|------|
| HP満タン時 | hp_full | HP100% |
| HP半分以上 | hp_half_or_more | HP≥50% |
| HP半分以下 | hp_half_or_less | HP≤50% |

---

## 3. ダメージ倍率（内部処理）

| 表示名 | 内部倍率 | 用途 |
|--------|----------|------|
| 強化 | ×1.2〜1.3 | 攻撃強化 |
| 大幅強化 | ×1.5 | 条件付き攻撃強化 |
| 軽減 | ×0.85〜0.9 | 被ダメージ軽減 |
| 大幅軽減 | ×0.5 | 被ダメージ大幅軽減 |

## 4. 種族パッシブと種族アビリティの違い

### 4.1 種族パッシブ（効果1つのみ）

種族パッシブは、その種族のモンスター全員が**自動で持っている**特性。選択不要。**アビリティ封じでも無効化されない**。

| 種族 | パッシブ名 | 効果 | 説明文 |
|------|-----------|------|--------|
| human | 装備の達人 | 装備枠+1 | 装備を1つ多く付けられる |
| angel | 天の恵み | 毎ターンHP8回復 | 毎ターン終了時、HPが8回復する |
| demon | 略奪本能 | 攻撃命中時バフ1つ解除 | 攻撃を当てると相手の強化を1つ解除 |
| dragon | 竜の威厳 | 状態異常は1種類まで | 同時に2つ以上の状態異常にならない（上書き方式） |
| mechanoid | 機械の体 | 状態異常-1ターン | 状態異常の持続が1ターン短くなる |
| spirit | 霊気 | 状態異常+1ターン | 与える状態異常が1ターン長く続く |
| mutant | 変異体 | 技スロット+1 | 技を1つ多くセットできる |

### 4.2 種族アビリティ（パッシブとは別効果）

| 種族 | アビリティ名 | 効果 |
|------|-------------|------|
| human | 適応力 | 同じ技を2回受けると、その技のダメージ半減 |
| angel | 聖なるオーラ | 回復技の効果+8 |
| demon | 魂喰らい | 相手を倒した時、全ステータス↑1 |
| dragon | 竜鱗 | 攻撃技が強化される（×1.3） |
| mechanoid | 電磁シールド | 雷属性のダメージを半減 |
| spirit | 霊体 | 攻撃技のダメージを軽減（×0.8） |
| mutant | 変幻の魂 | 被ダメージ時、最も低いステータス↑1 |

## 8. アビリティの発動タイミング

### 8.1 基本トリガー

| タイミング | 説明 | 例 |
|-----------|------|-----|
| バトル開始時 | 常時発動型が有効化 | 守りの構え、背水の陣 |
| 登場時 (on_entry) | 場に出た瞬間 | 畏怖の波動、支配の瞳 |
| ターン終了時 (turn_end) | ターン処理の最後 | 再生細胞、天の恵み |
| 被ダメージ時 (on_attack_received) | ダメージを受けた時 | 反射の鎧、帯電被膜 |
| 攻撃命中時 (on_hit) | 攻撃が当たった時 | 強欲の力（旧仕様） |
| 瀕死時 (on_death) | HP0になった時 | 女神の祝福、最期の呪詛 |
| 撃破時 (on_kill) | 相手を倒した時 | 搾取、魂喰らい |
| 交代時 (on_switch) | 交代で退場する時 | 置き土産、聖なる守護 |
| 常時 (passive) | 常に効果が有効 | 竜鱗、守りの構え |

### 8.2 複合トリガー

| トリガー | 発動条件 | 動作 |
|----------|---------|------|
| **on_entry_and_hit** | 登場時 + 攻撃命中時 | 登場時に効果1発動、攻撃命中時に効果2発動（1T1回制限可） |

**使用例**: 貪欲マモン「強欲の力」
```json
{
  "trigger": "on_entry_and_hit",
  "entry_buff": {"attack": 1},
  "stat_change": {"attack": 1},
  "hit_limit_per_turn": 1
}
```

## 10. アビリティ無効化

### 10.1 グラムの「アビリティ封じ」【v17.0改修】

- 効果: 相手のアビリティを無効化
- 発動: **登場時から2ターン**【v17.0変更: 3T→2T】
- 解除: **2ターン経過後に自動解除**

### 10.2 無効化されるもの

- 常時発動型: 効果が一時停止
- 条件発動型: 条件を満たしても発動しない
- 復活系/耐え系: 発動しない
- 種族アビリティ: 無効化される

### 10.3 無効化されないもの

- **種族パッシブ**（装備枠+1、技枠+1、HP自動回復、竜の威厳、状態異常短縮/延長など）
