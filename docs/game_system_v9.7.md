## 1. バトル選出ルール

### 1.1 基本ルール

| 要素 | 内容 |
|------|------|
| パーティ編成 | 5体 |
| バトル中に出せる数 | 最大3体 |
| カテゴリ制限 | なし |

### 1.2 ポイント

- バトル中に「どの3体を使うか」が決まる
- 選出した3体の中ではサイクル（交代→再登場）が可能
- 残り2体は「使わなかった保険」
- 交代時にバフ/デバフは解除、状態異常は引き継ぐ

## 2. レベルシステム

### 2.1 基本ルール

| 要素 | 内容 |
|------|------|
| 最大レベル | 50 |
| PvPバトル時 | **Lv50固定**でステータス計算（対戦の公平性） |
| 冒険・PvE時 | 実際のレベル・振り分けでステータス計算 |

### 2.2 HP成長

| 項目 | 値 |
|------|-----|
| Lv1時HP | 52 |
| Lv50時HP | **150（固定）** |
| 成長値 | +2/Lv |
| 計算式 | `HP = 52 + (Lv - 1) × 2` |

### 2.3 振り分けポイント獲得

| 項目 | 値 |
|------|-----|
| 計算式 | `振り分けpt = Lv × 4` |
| Lv50 | 200pt |

### 2.4 経験値テーブル

**累計100,000でLv50到達**

**計算式**: `次Lvまで = Lv × 80 + 10`

| レベル | 次Lvまで | 累計 | 備考 |
|--------|---------|------|------|
| 1→2 | 90 | 90 | |
| 5→6 | 410 | 1,250 | 序盤 |
| 10→11 | 810 | 4,050 | |
| 15→16 | 1,210 | 9,100 | |
| 20→21 | 1,610 | 16,100 | 中盤 |
| 25→26 | 2,010 | 25,100 | |
| 30→31 | 2,410 | 36,150 | |
| 35→36 | 2,810 | 49,200 | |
| 40→41 | 3,210 | 64,200 | 終盤 |
| 45→46 | 3,610 | 81,250 | |
| 49→50 | 3,930 | 100,000 | 最大 |

### 2.5 ステージモンスターからの獲得経験値

| エリア | モンスターLv帯 | 1体あたり | 1ステージ（5体） | 1Lv上げに必要な周回 |
|--------|--------------|----------|-----------------|-------------------|
| 1-2 | Lv5-10 | 30-50 | 150-250 | 2-3周 |
| 3-4 | Lv15-20 | 80-120 | 400-600 | 3周 |
| 5-6 | Lv25-30 | 150-200 | 750-1,000 | 3周 |
| 7-8 | Lv35-40 | 250-320 | 1,250-1,600 | 2-3周 |
| 9-10 | Lv45-50 | 400-500 | 2,000-2,500 | 2周 |
| ボス | - | 800-3,000 | - | ボーナス |

**設計意図**:
- 1ステージクリアで適正レベルが0.3-0.5レベル上がる
- 3-5ステージで1レベル上昇（周回は必要だが苦行にならない）
- 冒険だけでLv1→50 = 約40-50ステージ分

## 3. ステータスシステム

### 3.1 基本構造

| 要素 | 内容 |
|------|------|
| HP | **150固定**（PvP時） |
| ステータス種類 | **攻撃、防御、魔力、魔防、速さ（5種）** |
| 基礎値合計 | **238〜252** |
| 基礎値範囲 | **10〜97** |
| 振り分けポイント | **200pt**（Lv50時） |
| 振り分け上限 | **各60** |
| 個体値補正 | **±12固定値** |

### 3.2 ステータス計算式

```
最終ステータス = 基礎値 + 振り分け + 個体値補正
```

### 3.3 ステータス例外規定（v15.1追加）

| 項目 | 内容 |
|------|------|
| 例外許容 | 一部モンスターは基礎値合計238〜252の範囲を超える場合がある |
| 理由 | Tier表作成時に弱いモンスターの救済措置としてステータス強化を実施 |
| 例外モンスター | タロス(237)、ジェイド(250)、ビットマン(256)、シーラ(252)、マモン(265)、セラ(265)、モフ(268) |
| 設計思想 | 「カテゴリ＝強さ」ではないため、一般種でも幻種以上のステータスを許容 |

|------|--------|

| 同時付与上限 | **2種類まで** |
| ダメージ基準 | **最大HP** |
| 端数処理 | **切り上げ** |

### 7.2 状態異常一覧（v9.8改修）

| 状態異常 | 効果 | 持続 | HP150時 |
|----------|------|------|---------|
| 毒 | 毎T 最大HP**4%**ダメージ | 3T | **6/T** |
| 火傷 | 毎T 最大HP3%ダメージ + 攻撃**-10%** | 3T | 5/T |
| 瘴気 | 毎T 最大HP3%ダメージ + 魔力**-10%** | 3T | 5/T |
| 麻痺 | 素早さ**-20%** | 3T | - |
| 拘束 | **このターンと次のターン**交代不可 | **2T** | - |
| 封印 | 最高コスト技使用不可 | 2T | - |
| 魅了 | 被ダメージ**+15%** | 2T | - |

### 7.3 拘束の詳細（v9.8改修）

拘束は「**このターンと次のターン交代不可**」になる。

```
T1: 拘束付与（残り2T）
T1: 交代不可（このターン）
T2: 交代不可（次のターン）
T2終了時: 拘束解除
```

## 10. 種族と種族特性（v9.8改修）

### 10.1 種族パッシブ（効果1つのみ）

種族パッシブは**効果1つのみ**。シンプルで分かりやすい設計。
**グラムの「アビリティ封じ」でも無効化されない。**

| 種族 | パッシブ名 | 効果 | 説明文 |
|------|-----------|------|--------|
| human | 装備の達人 | 装備枠+1 | 装備を1つ多く付けられる |
| angel | 天の恵み | 毎ターンHP5%回復 | 毎ターン終了時、HPが少し回復する |
| demon | 略奪本能 | 攻撃命中時バフ1つ奪う | 攻撃を当てると相手の強化を1つ奪う |
| **dragon** | **竜の威厳** | **状態異常は1種類まで** | **同時に2つ以上の状態異常にならない** |
| mechanoid | 機械の体 | 状態異常-1ターン | 状態異常の持続が1ターン短くなる |
| spirit | 霊気 | 状態異常+1ターン | 与える状態異常が1ターン長く続く |
| mutant | 変異体 | 技スロット+1 | 技を1つ多くセットできる |

**dragon「竜の威厳」の詳細:**
- 既に状態異常を受けている状態で新たな状態異常を受けると、**新しい状態異常で上書き**
- 例: 火傷中に麻痺を受ける → 火傷が消え、麻痺状態に変化

### 10.2 種族アビリティ（パッシブとは別効果）

| 種族 | アビリティ名 | 効果 |
|------|-------------|------|
| human | 適応力 | 同じ技を2回受けると×0.5 |
| angel | 聖なるオーラ | 回復技×1.3 |
| demon | 魂喰らい | 相手を倒した時、全ステ↑1 |
| dragon | 竜鱗 | 攻撃技×1.3 |
| mechanoid | 電磁シールド | 雷属性×0.5 |
| spirit | 霊体 | 攻撃技（物理）×0.8 |
| mutant | 変幻の魂 | 被ダメ時ランダムステ↑1 |

## 15. 強化結晶システム（v9.9新規）

モンスターのステータスを追加で強化できる育成要素。

### 15.1 基本ルール

| 強化上限 | 合計**+30**まで |
| 各ステ上限 | 各**+10**まで |
| 上昇値 | 1結晶 = +1 |
| 適用タイミング | いつでも可能 |

### 15.2 結晶種類

| アイテム | 効果 | 個別上限 |
|---------|------|----------|
| 攻撃の結晶 | 攻撃+1 | +10まで |
| 防御の結晶 | 防御+1 | +10まで |
| 魔力の結晶 | 魔力+1 | +10まで |
| 魔防の結晶 | 魔防+1 | +10まで |
| 速さの結晶 | 速さ+1 | +10まで |
| 万能の結晶 | 任意+1 | 各上限内 |

### 15.3 入手方法

| 方法 | 詳細 |
|------|------|
| ドロップ | 冒険・ボス戦で低確率ドロップ |
| 課金 | ショップで購入可能 |
| イベント | 報酬として入手 |
| バトルパス | パス報酬として入手 |

### 15.4 計算式への影響

```
最終ステータス = 基礎値 + 振り分け + 個体値補正 + 結晶強化値
```

### 15.5 設計意図

- **差別化要素**: 同じモンスターでも育成度で差が出る
- **弱点補強**: 苦手なステータスを補う選択肢
- **特化強化**: 得意なステータスをさらに伸ばす選択肢
- **ガチャ外課金要素**: ガチャ以外の課金で強化可能

## 18. 素材変換システム（v12.5新規）

低レア度の基本素材を高レア度の基本素材に変換するシステム。

### 18.1 基本ルール

| 項目 | 内容 |
|------|------|
| 対象素材 | **基本素材のみ**（10種） |
| 変換方向 | **低レア → 高レア**のみ（逆変換不可） |
| 変換レート | **段階的**（下表参照） |
| ゴールド消費 | **0G**（無料） |

### 18.2 変換レート

| 変換 | レート | 理由 |
|------|--------|------|
| ★1→★2 | **5:1** | ドロップ率差が小さい、序盤救済 |
| ★2→★3 | **8:1** | 中盤への橋渡し |
| ★3→★4 | **10:1** | 上位素材への壁 |
| ★4→★5 | **15:1** | 最上位素材の価値維持 |

### 18.3 変換レシピ

| 入力素材 | 必要数 | 出力素材 | 獲得数 |
|---------|--------|---------|--------|
| 粗鉄鉱（★1） | 5 | 魔鉱石（★2） | 1 |
| 獣の骨（★1） | 5 | 古代樹の枝（★2） | 1 |
| 魔鉱石（★2） | 8 | 精霊石（★3） | 1 |
| 古代樹の枝（★2） | 8 | 星屑の砂（★3） | 1 |
| 精霊石（★3） | 10 | ミスリル（★4） | 1 |
| 星屑の砂（★3） | 10 | 霊魂の欠片（★4） | 1 |
| ミスリル（★4） | 15 | オリハルコン（★5） | 1 |
| 霊魂の欠片（★4） | 15 | 混沌の核（★5） | 1 |

### 18.4 変換の活用例

**例1: エリア1-2の周回で★3素材を入手**
```
粗鉄鉱 x40 → 魔鉱石 x8 → 精霊石 x1
```

**例2: 余った★2素材を★4に昇華**
```
魔鉱石 x80 → 精霊石 x10 → ミスリル x1
```

### 18.5 変換不可の素材

以下の素材は変換**不可**:

| カテゴリ | 理由 |
|---------|------|
| カテゴリ素材 | ボス撃破でしか入手できない希少素材 |
| 属性素材 | 該当エリアでしか入手できない限定素材 |
| 種族素材 | 特定種族を倒す目的を維持するため |
| レアドロップ | 希少種/幻種/ボス限定の価値を維持 |
| 結晶 | 被りモンスター変換の価値を維持 |
| ★5基本素材 | 最上位素材のため変換先がない |

### 18.6 設計意図

1. **素材の無駄をなくす**: 低エリアの周回でも上位素材に繋がる
2. **ゴールドインフレ防止**: 売却ではなく変換に誘導
3. **周回の価値向上**: どのエリアでも素材集めが有意義に
4. **長期目標の設定**: 大量変換で最上位素材を目指すやりこみ要素

### 18.7 変換効率

| 目標素材 | 必要な★1素材 | 周回目安（エリア1-2） |
|---------|-------------|---------------------|
| ★2素材 x1 | 5個 | 約15分 |
| ★3素材 x1 | 40個 | 約2時間 |
| ★4素材 x1 | 400個 | 約20時間 |
| ★5素材 x1 | 6,000個 | 約300時間 |

※ 上位エリアで直接入手する方が効率的。変換は「余った素材の活用」が主目的。
