## 1. バトル選出ルール

### 1.1 基本ルール

| 要素 | 内容 |
|------|------|
| パーティ編成 | 5体 |
| バトル中に出せる数 | 最大3体 |
| カテゴリ制限 | なし |

### 1.2 ポイント

- バトル中に「どの3体を使うか」が決まる
- 選出した3体の中ではサイクル（交代→再登場）が可能
- 残り2体は「使わなかった保険」
- 交代時にバフ/デバフは解除、状態異常は引き継ぐ

## 2. レベルシステム

### 2.1 基本ルール

| 要素 | 内容 |
|------|------|
| 最大レベル | 50 |
| PvPバトル時 | **Lv50固定**でステータス計算（対戦の公平性） |
| 冒険・PvE時 | 実際のレベル・振り分けでステータス計算 |

### 2.2 HP成長

| 項目 | 値 |
|------|-----|
| Lv1時HP | 52 |
| Lv50時HP | **150（固定）** |
| 成長値 | +2/Lv |
| 計算式 | `HP = 52 + (Lv - 1) × 2` |

### 2.3 振り分けポイント獲得

| 項目 | 値 |
|------|-----|
| 計算式 | `振り分けpt = Lv × 4` |
| Lv50 | 200pt |

## 3. ステータスシステム

### 3.1 基本構造

| 要素 | 内容 |
|------|------|
| HP | **150固定**（PvP時） |
| ステータス種類 | **攻撃、防御、魔力、魔防、速さ（5種）** |
| 基礎値合計 | **238〜252** |
| 基礎値範囲 | **10〜97** |
| 振り分けポイント | **200pt**（Lv50時） |
| 振り分け上限 | **各60** |
| 個体値補正 | **±12固定値** |

### 3.2 ステータス計算式

```
最終ステータス = 基礎値 + 振り分け + 個体値補正
```

## 6. 技プールと奥義

### 6.1 基本ルール

| 項目 | 内容 |
|------|------|
| 技スロット | 基本4つ（mutant種族は5つ） |

### 6.2 奥義（v9.8改修）

- **バトル中1回のみ**（パーティ全体で1回）
- コストはモンスターごとに異なる（**3-6**）
- **同コストなら同等の強さ（カテゴリ無関係）**

### 6.3 コスト別基準

| コスト | 威力 | 追加効果 |
|--------|------|---------|
| 3 | C | 強力な1つ |
| 4 | B | 中程度の1つ |
| 5 | A | 軽めの1つ |
| 6 | S | なし or 軽め |

## 7. 状態異常システム（v9.8改修）

### 7.1 基本ルール

| 項目 | ルール |
|------|--------|
| 付与確率 | 100%確定 |
| 同時付与上限 | **2種類まで** |
| ダメージ基準 | **最大HP** |
| 端数処理 | **切り上げ** |

### 7.2 状態異常一覧（v9.8改修）

| 状態異常 | 効果 | 持続 | HP150時 |
|----------|------|------|---------|
| 毒 | 毎T 最大HP**4%**ダメージ | 3T | **6/T** |
| 火傷 | 毎T 最大HP3%ダメージ + 攻撃**-10%** | 3T | 5/T |
| 瘴気 | 毎T 最大HP3%ダメージ + 魔力**-10%** | 3T | 5/T |
| 麻痺 | 素早さ**-20%** | 3T | - |
| 拘束 | **このターンと次のターン**交代不可 | **2T** | - |
| 封印 | 最高コスト技使用不可 | 2T | - |
| 魅了 | 被ダメージ**+15%** | 2T | - |
| 恐怖 | 与ダメージ**-15%** | 2T | - |
| 悪夢 | 毎T 最大HP3%ダメージ + 回復効果**-50%** | 3T | 5/T |

### 7.3 拘束の詳細（v9.8改修）

拘束は「**このターンと次のターン交代不可**」になる。

```
T1: 拘束付与（残り2T）
T1: 交代不可（このターン）
T2: 交代不可（次のターン）
T2終了時: 拘束解除
```

## 10. 種族と種族特性（v9.8改修）

### 10.1 種族パッシブ（効果1つのみ）

種族パッシブは**効果1つのみ**。シンプルで分かりやすい設計。
**グラムの「アビリティ封じ」でも無効化されない。**

| 種族 | パッシブ名 | 効果 | 説明文 |
|------|-----------|------|--------|
| human | 装備の達人 | 装備枠+1 | 装備を1つ多く付けられる |
| angel | 天の恵み | 毎ターンHP5%回復 | 毎ターン終了時、HPが少し回復する |
| demon | 略奪本能 | 攻撃命中時バフ1つ奪う | 攻撃を当てると相手の強化を1つ奪う |
| **dragon** | **竜の威厳** | **状態異常は1種類まで** | **同時に2つ以上の状態異常にならない** |
| mechanoid | 機械の体 | 状態異常-1ターン | 状態異常の持続が1ターン短くなる |
| spirit | 霊気 | 状態異常+1ターン | 与える状態異常が1ターン長く続く |
| mutant | 変異体 | 技スロット+1 | 技を1つ多くセットできる |

**dragon「竜の威厳」の詳細:**
- 既に状態異常を受けている状態で新たな状態異常を受けると、**新しい状態異常で上書き**
- 例: 火傷中に麻痺を受ける → 火傷が消え、麻痺状態に変化

### 10.2 種族アビリティ（パッシブとは別効果）

| 種族 | アビリティ名 | 効果 |
|------|-------------|------|
| human | 適応力 | 同じ技を2回受けると×0.5 |
| angel | 聖なるオーラ | 回復技×1.3 |
| demon | 魂喰らい | 相手を倒した時、全ステ↑1 |
| dragon | 竜鱗 | 攻撃技×1.3 |
| mechanoid | 電磁シールド | 雷属性×0.5 |
| spirit | 霊体 | 攻撃技（物理）×0.8 |
| mutant | 変幻の魂 | 被ダメ時ランダムステ↑1 |
