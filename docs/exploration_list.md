## 1. 探検システム概要

### 1.1 基本ルール

| 項目 | 内容 |
|------|------|
| 解放条件 | 各エリアのボス撃破でそのエリアの探検が解放 |
| 同時派遣数 | **最大3箇所** |
| 派遣モンスター数 | 各探検先に**1体** |
| 派遣中の制限 | 派遣モンスターは**バトル使用不可** |
| 派遣時間 | **1時間 / 4時間 / 8時間** から選択 |

### 1.2 派遣時間ボーナス

| 派遣時間 | ボーナス倍率 | メリット |
|---------|------------|---------|
| 1時間 | ×1.0（なし） | こまめに回収できる |
| 4時間 | ×1.1（+10%） | バランス型 |
| 8時間 | ×1.2（+20%） | 放置向け、最高効率 |

### 1.3 報酬の計算方法

```
最終報酬 = 基本報酬/h × 派遣時間 × ボーナス倍率

例: エリア10に8時間派遣
経験値 = 250 × 8 × 1.2 = 2,400
ゴールド = 200 × 8 × 1.2 = 1,920
```

※ 素材は確率判定のため、端数は「30%で1個」のように処理

## 探検戦略ガイド

### 初心者向け（エリア1-4解放）

1. **8時間派遣を基本**に設定
2. 複数エリアに分散派遣で素材バランス確保
3. ゴールドは冒険で稼ぎ、素材は探検で集める

### 中級者向け（エリア5-7解放）

1. **カテゴリ素材**を意識した探検先選択
2. 必要な装備に合わせて派遣先を調整
3. 余裕があれば高レベルエリアに挑戦

### 上級者向け（エリア8-10解放）

1. **伝説の核**を狙って混沌の終焉を重点派遣
2. ★4-5素材を効率的に収集
3. 長期目標（伝説装備）に向けた計画的派遣
