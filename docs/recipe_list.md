## 1. レシピシステム概要

### 1.1 錬成の基本ルール

- 錬成画面で装備を選択し、必要素材とゴールドを消費して作成
- レシピは最初から全て閲覧可能（素材が足りなくても確認できる）
- 同じ装備は何度でも錬成可能
- 錬成に失敗はない（素材があれば確定で作成）

### 1.2 必要素材の構成

| 要素 | 説明 |
|------|------|
| ゴールド | 錬成費用（レア度に応じて増加） |
| 基本素材 | レア度に対応した鉱石類（2種類の組み合わせ） |
| カテゴリ素材 | 装備カテゴリに対応した専用素材 |
| 属性素材 | 属性系装備のみ必要 |
| 種族素材 | 種族キラー装備に必要 |
| レアドロップ | 一部の強力な装備に必要 |
| 結晶 | rare以上の装備に必要 |

### 1.3 レシピの差別化

- 同じレア度でも装備によって基本素材の組み合わせが異なる
- 例: 「粗鉄鉱x3 + 獣の骨x2」「粗鉄鉱x2 + 獣の骨x3」など
- 特定の装備は種族素材やレアドロップが必要

## 設計メモ

### レシピ差別化の意図

1. **物理系装備**: 鉱石（粗鉄鉱、魔鉱石、精霊石）多め
2. **魔法系装備**: 有機素材（獣の骨、古代樹、星屑）多め
3. **属性装備**: 属性イメージに合わせた配分

### 特殊素材の設計意図

**覇者の勲章が必要な装備**（8種）:
- 状態異常系最上位（浄化の護符、状態反転の札）
- 強力な戦術系（時空の砂時計、起死回生の薬など）
- legendary_special全種

**種族素材が必要な装備**（7種）:
- 種族キラー装備全種
- 該当種族を狩る目的が生まれる

**幻種の証が必要な装備**（10種）:
- legendary装備全種
- 幻種との遭遇価値を高める

### やりこみ要素としての設計

- 全装備コンプには膨大な素材が必要
- 特に種族素材（5%ドロップ）は周回が必要
- 幻種の証（10%ドロップ、幻種限定）は最高難度の収集要素
