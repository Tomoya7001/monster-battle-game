## 📋 基本ルール

### 装備スロット
| 種族 | スロット数 | 備考 |
|:-----|:----------:|:-----|
| ヒューマン | 2枠 | 種族パッシブ |
| その他全種族 | 1枠 | - |

### 装備ルール
| ルール | 内容 |
|:-------|:-----|
| 効果数 | **最大1つまで** |
| 同一装備制限 | **同じ装備は2つセット不可**（ヒューマンでも） |
| ステータス反映 | 装備中はステータスに即時反映 |
| 装備解除時 | 元のステータスに戻る |
| レアリティ | コモン / レア / エピック / レジェンダリー |

### HP条件（3種類のみ）
| 条件名 | 内部値 | 説明 |
|:-------|:-------|:-----|
| HP満タン時 | hp_full | HP = 100% |
| HP半分以上 | hp_half_or_more | HP ≥ 50% |
| HP半分以下 | hp_half_or_less | HP ≤ 50% |

### 倍率表示（内部処理・非表示）
| 表示名 | 内部倍率 | 用途 |
|:-------|:--------:|:-----|
| 威力アップ | ×1.2 | 属性技強化 |
| 少し上がる | ×1.15 | 軽微な強化 |
| 強化 | ×1.3 | 条件付き攻撃強化 |
| 軽減 | ×0.8 | 被ダメージ軽減 |

# 🔧 実装時の注意事項

## ステータス反映処理
```
装備時:
  1. 装備のステータス効果を取得
  2. モンスターの現在ステータスに加算
  3. UI上のステータス表示を更新

解除時:
  1. 装備のステータス効果を取得
  2. モンスターの現在ステータスから減算
  3. UI上のステータス表示を更新
```

## 同一装備チェック
```
装備セット時:
  1. 既に同じ装備がセットされているか確認
  2. セット済みの場合はエラー表示
  3. 「同じ装備は2つセットできません」
```

## 種族制限チェック
```
装備セット時:
  1. 装備の種族制限を確認
  2. モンスターの種族と照合
  3. 制限に該当する場合はエラー表示
  4. 「この装備は〇〇種族には装備できません」
```

# 🚫 採用しなかった装備案

| 案 | 理由 |
|:---|:-----|
| 速さ+15 | 先行有利すぎ、キャラの個性が消える |
| HP+30（低レアリティ版） | レアリティで効果量を変えない方針 |
| 〇〇%上昇 | %表示は避ける方針 |
| 複数効果（回復+被ダメ軽減等） | 効果は1つまでの原則 |
| ★2限定装備 | ★システム廃止のため |
| 重圧のオーラ（相手コスト+1） | 強すぎる |
| 透明のマント（1T無敵） | 起点作りが強すぎる |
