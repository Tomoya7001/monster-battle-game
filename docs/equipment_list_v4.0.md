## 📋 基本ルール

### 装備スロット
| 種族 | スロット数 | 備考 |
|:-----|:----------:|:-----|
| ヒューマン | 2枠 | 種族パッシブ |
| その他全種族 | 1枠 | - |

### 装備ルール
| ルール | 内容 |
|:-------|:-----|
| 効果数 | **最大1つまで** |
| 同一装備制限 | **同じ装備は2つセット不可**（ヒューマンでも） |
| ステータス反映 | 装備中はステータスに即時反映 |
| 装備解除時 | 元のステータスに戻る |
| レアリティ | コモン / レア / エピック / レジェンダリー |
| 運要素 | **完全排除**（ランダム効果禁止） |

### HP条件（3種類のみ）
| 条件名 | 内部値 | 説明 |
|:-------|:-------|:-----|
| HP満タン時 | hp_full | HP = 100% |
| HP半分以上 | hp_half_or_more | HP ≥ 50% |
| HP半分以下 | hp_half_or_less | HP ≤ 50% |

### 倍率表示（内部処理・非表示）
| 表示名 | 内部倍率 | 用途 |
|:-------|:--------:|:-----|
| 威力アップ | ×1.2 | 属性技強化 |
| 少し上がる | ×1.15 | 軽微な強化 |
| 強化 | ×1.3 | 条件付き攻撃強化 |
| 軽減 | ×0.8 | 被ダメージ軽減 |
