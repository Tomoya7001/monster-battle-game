## 適用方法

### 方法1: VS Codeの検索置換（推奨）

1. VS Codeで`assets/data`フォルダを開く
2. `Ctrl+Shift+H`（検索と置換）
3. 以下の順で置換を実行：

```
"突進" → "猛進"
"威嚇" → "威圧"
"挑発" → "煽動"
"生命賦活" → "活力の息吹"
"追い風" → "順風"
"月光" → "癒月"
"光壁" → "光の障壁"
"羽休め" → "翼の休息"
"居合斬り" → "抜刀術"
```

### 方法2: シェルスクリプト

```bash
cd assets/data

# 各ファイルで置換
for file in monster_masters_data_*.json skill_masters_*.json; do
  sed -i 's/"突進"/"猛進"/g' "$file"
  sed -i 's/"威嚇"/"威圧"/g' "$file"
  sed -i 's/"挑発"/"煽動"/g' "$file"
  sed -i 's/"生命賦活"/"活力の息吹"/g' "$file"
  sed -i 's/"追い風"/"順風"/g' "$file"
  sed -i 's/"月光"/"癒月"/g' "$file"
  sed -i 's/"光壁"/"光の障壁"/g' "$file"
  sed -i 's/"羽休め"/"翼の休息"/g' "$file"
  sed -i 's/"居合斬り"/"抜刀術"/g' "$file"
done
```
