| HPを多く回復 | 50 | heal_fixed_50 |
| HPを大きく回復 | 80 | heal_fixed_80 |
| HPを全回復 | max | heal_full |

**設計意図:**
- HP150固定（装備で200）の設計に対応
- 装備でHPが増えても回復量は一定（バランス維持）
- プレイヤーに計算させない直感的な表現
