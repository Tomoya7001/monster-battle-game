# 技システム仕様書

**作成日**: 2025-12-16  
**バージョン**: v1.0  
**ステータス**: Phase1 確定版

---

## 1. 概要

技はモンスターがバトル中に使用するアクティブスキル。コストを消費して発動し、ダメージや特殊効果を与える。

---

## 2. 技の基本構造

### 2.1 データ構造

| フィールド | 型 | 説明 |
|-----------|-----|------|
| skill_id | string | 技ID（一意） |
| name | string | 技名 |
| element | string | 属性（fire/water/thunder/wind/earth/light/dark/none） |
| cost | int | 消費コスト（1-6） |
| power | string | 威力ランク（D/C/B/A/S/SS/SSS） |
| damage_type | string | ダメージ種別（physical/magical/none） |
| accuracy | int | 命中率（%） |
| priority | int | 優先度（-1〜+3） |
| effects | array | 特殊効果リスト |
| description | string | 説明文 |

### 2.2 技の分類

```
【A】ダメージのみ（特殊効果なし）
【B】サポートのみ（ダメージなし）
【C】ダメージ + 特殊効果
【D】奥義（バトル中1回のみ）
```

## 3. 威力ランクシステム

### 3.1 威力倍率対応表

| ランク | 威力倍率 | コスト目安 | 用途 |
|--------|----------|-----------|------|
| D | 0.30 | 1 | 小技、先制技 |
| C | 0.55 | 1-2 | 通常技 |
| B | 0.80 | 2-3 | 中威力技 |
| A | 1.10 | 3-4 | 強技 |
| S | 1.50 | 4-5 | 奥義 |
| SS | 2.00 | 5-6 | 強奥義 |
| SSS | 2.50-3.00 | 6 | 最強奥義 |

### 3.2 ダメージ計算式

```
ダメージ = 攻撃側ステ × 技威力倍率 × タイプ相性 × 防御補正 × 乱数 × クリティカル

【各要素】
- 攻撃側ステ: 物理技→攻撃 / 魔法技→魔力
- タイプ相性: 有利×1.8 / 等倍×1.0 / 不利×0.55
- 防御補正: 50 / (50 + 防御側ステ)
- 乱数: 0.85〜1.00
- クリティカル: 発生時×1.5 / 通常×1.0
```

### 3.3 ダメージ計算例

```
【例】テュール（攻撃150）が業炎（威力A=1.10）で水属性（防御80）を攻撃

ダメージ = 150 × 1.10 × 0.55 × (50/130) × 0.92 × 1.0
        = 150 × 1.10 × 0.55 × 0.385 × 0.92
        ≒ 32ダメージ
```

## 4. コストシステム

### 4.1 基本ルール

| 項目 | 値 |
|------|-----|
| 初期コスト | 4 |
| 最大コスト | 7 |
| 毎ターン回復 | +2 |

### 4.2 コスト別技の目安

| コスト | 威力 | 使用頻度 |
|--------|------|---------|
| 1 | D-C | 毎ターン使用可 |
| 2 | C-B | 毎ターン使用可 |
| 3 | B-A | 2ターンに1回 |
| 4 | A-S | 奥義級 |
| 5 | S-SS | 強奥義級 |
| 6 | SS-SSS | 最強奥義 |

## 5. 優先度システム

### 5.1 優先度一覧

| 優先度 | 分類 | 技例 |
|--------|------|------|
| +3 | 交代時特殊 | 追撃（相手交代時のみ） |
| +2 | 強先制技 | 神速斬、閃光撃 |
| +1 | 通常先制技 | 影縫い、激流撃、疾雷、真空波 |
| 0 | 通常技 | 全ての通常技、防護壁 |
| -1 | 後攻技 | カウンター、魔力返し、報復 |

### 5.2 行動順決定

```
1. 優先度が高い技を使用した方が先制
2. 同優先度の場合は素早さで判定
3. 同優先度・同素早さの場合はランダム（50%）
```

### 6.1 バフ・デバフ

| 段階 | 倍率 |

### 6.2 状態異常付与

| 状態異常 | 効果 | 持続 |
|----------|------|------|
| 毒 | 毎T HP2%ダメージ | 3T |
| 火傷 | 毎T HP3%ダメージ + 攻撃-5% | 3T |
| 瘴気 | 毎T HP3%ダメージ + 魔力-5% | 3T |
| 麻痺 | 素早さ-15% | 3T |

| 封印 | 最高コスト技使用不可 | 2T |

| 悪夢 | 毎T魔力3%ダメージ + 回復効果-50% | 3T |

### 6.3 回復効果

| 効果 | 説明 |
|------|------|
| HP回復 | 最大HPの○%回復 |
| 状態異常解除 | 状態異常を解除 |
| バフ解除 | 相手のバフを解除 |

### 6.4 その他効果

| 効果 | 説明 |
|------|------|
| 防護 | そのターン全ダメージ無効（連続使用不可） |
| 吸収 | 与ダメージの○%回復 |
| 反動 | 与ダメージの○%を自分も受ける |
| 設置 | フィールドに効果を設置 |
| 強制交代 | 相手を強制的に交代させる |

## 7. クリティカルシステム

### 7.1 発生率

| 条件 | 発生率 |
|------|--------|
| 通常 | 6% |
| 急所ランク+1 | 12% |
| 急所ランク+2 | 50% |
| 急所ランク+3以上 | 100% |

### 7.2 クリティカル時の効果

- ダメージ×1.5
- 相手の防御バフを無視

## 9. 奥義システム

### 9.1 基本ルール

- バトル中1回のみ使用可能（パーティ全体で1回）
- コストはモンスターごとに異なる（3-6）
- カテゴリに関係なく、同コストなら同等の強さ

### 9.2 奥義の特徴

| コスト | 威力目安 | 追加効果 |
|--------|---------|---------|
| 3 | S | 小規模な追加効果 |
| 4 | S-SS | 中規模な追加効果 |
| 5 | SS | 大規模な追加効果 |
| 6 | SSS | 最大級の追加効果 |
