# 技システム仕様書 v2.0

**作成日**: 2025-12-16  
**最終更新**: 2025-12-20  
**バージョン**: v2.0（バランス改修版）  
**ステータス**: Phase1 確定版

---

## 1. 概要

技はモンスターがバトル中に使用するアクティブスキル。コストを消費して発動し、ダメージや特殊効果を与える。

---

## 2. 技の基本構造

### 2.1 データ構造

| フィールド | 型 | 説明 |
|-----------|-----|------|
| skill_id | string | 技ID（一意） |
| name | string | 技名 |
| element | string | 属性（fire/water/thunder/wind/earth/light/dark/none） |
| cost | int | 消費コスト（1-6） |
| power | string | 威力ランク（D/C/B/A/S） |
| damage_type | string | ダメージ種別（physical/magical/none） |
| accuracy | int | 命中率（%）※通常100% |
| priority | int | 優先度（-1〜+3） |
| effects | array | 特殊効果リスト（**最大1つ**） |
| description | string | 説明文（**1行で収まる長さ**） |

### 2.2 技の分類

```
【A】ダメージのみ（特殊効果なし）
【B】サポートのみ（ダメージなし）
【C】ダメージ + 特殊効果（効果は1つまで）
【D】奥義（バトル中1回のみ）
```

### 2.3 技設計のルール（v2.0シンプル化）

| ルール | 内容 |
|--------|------|
| 攻撃技 | 威力 + 追加効果（**最大1つ**） |
| 補助技 | 効果（**最大2つ**） |
| 説明文 | **1行で収まる長さ** |
| 複雑な効果 | **禁止**（○○時に△△など） |

## 4. コストシステム

### 4.1 基本ルール

| 項目 | 値 |
|------|-----|
| 初期コスト | 4 |
| 最大コスト | 7 |
| 毎ターン回復 | +2 |

### 4.2 コスト別技の目安

| コスト | 威力 | 使用頻度 |
|--------|------|---------|
| 1 | D-C | 毎ターン使用可 |
| 2 | C-B | 毎ターン使用可 |
| 3 | B | 2ターンに1回 |
| 4 | B-A | 奥義級 |
| 5 | A | 強奥義級 |
| 6 | S | 最強奥義 |

---

## 5. 優先度システム

### 5.1 優先度一覧

| 優先度 | 分類 | 技例 |
|--------|------|------|
| +3 | 交代時特殊 | 追撃（相手交代時のみ） |
| +2 | 強先制技 | 神速斬、閃光撃 |
| +1 | 通常先制技 | 影縫い、激流撃、疾雷、真空波 |
| 0 | 通常技 | 全ての通常技、防護壁 |
| -1 | 後攻技 | カウンター、魔力返し、報復 |

### 5.2 行動順決定

```
1. 優先度が高い技を使用した方が先制
2. 同優先度の場合は素早さで判定
3. 同優先度・同素早さの場合はランダム（50%）
```

## 6. 特殊効果一覧（v2.0シンプル化）

### 6.1 許可される効果（シンプルなもの）

**攻撃技の追加効果（1つまで）:**
- 状態異常確定
- ステータス変化（↑↓は1〜2段階）
- 先制
- 必中
- 急所率UP
- HP回復（与ダメの○%）
- 連続攻撃（2〜3回）

**補助技の効果（2つまで）:**
- HP回復（固定%）
- ステータス変化
- 状態異常付与
- 状態異常解除
- バフ解除
- 交代

### 6.2 バフ・デバフ

| 段階 | 倍率 |
|------|------|
| +3 | ×1.50 |
| +2 | ×1.33 |
| +1 | ×1.17 |
| 0 | ×1.00 |
| -1 | ×0.85 |
| -2 | ×0.75 |
| -3 | ×0.67 |

### 6.3 状態異常付与（v2.0改修）

| 状態異常 | 効果 | 持続 |
|----------|------|------|
| 毒 | 毎T HP**4%**ダメージ | 3T |
| 火傷 | 毎T HP3%ダメージ + 攻撃**-10%** | 3T |
| 瘴気 | 毎T HP3%ダメージ + 魔力**-10%** | 3T |
| 麻痺 | 素早さ**-20%** | 3T |
| 拘束 | **このターンと次のターン**交代不可 | **2T** |
| 封印 | 最高コスト技使用不可 | 2T |
| 魅了 | 被ダメージ**+15%** | 2T |
| 恐怖 | 与ダメージ**-15%** | 2T |
| 悪夢 | 毎T HP3%ダメージ + 回復効果**-50%** | 3T |

### 6.4 回復効果

| 効果 | 数値 |
|------|------|
| 少し回復 | 5% |
| 回復 | 20% |
| 大きく回復 | 30% |
| 全回復 | 100% |

### 6.5 HP吸収

| 効果 | 数値 |
|------|------|
| HP吸収 | 与ダメの20% |
| 大きく吸収 | 与ダメの25% |
