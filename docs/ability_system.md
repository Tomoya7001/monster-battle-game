# アビリティシステム仕様書

**作成日**: 2025-12-16  
**バージョン**: v1.0  
**ステータス**: Phase1 確定版

---

## 1. 概要

アビリティはモンスターが持つパッシブ能力。常時発動または特定条件で自動発動し、バトルを有利に進める。

---

## 2. アビリティの種類

### 2.1 分類

| 種類 | 説明 | 例 |
|------|------|-----|
| **種族特性** | 種族固有のパッシブ能力 | 装備マスター、変幻の魂 |
| **レア特性** | モンスター個別のパッシブ能力 | 女神の祝福、剣聖の極意 |

### 2.2 発動タイプ

| タイプ | 説明 |
|--------|------|
| 常時発動 | バトル中常に効果を発揮 |
| 条件発動 | 特定条件を満たすと発動 |
| 登場時発動 | 場に出た瞬間に発動 |
| 瀕死時発動 | HP0になった時に発動 |

### 3.1 全7種族の特性

| 種族 | 特性名 | 効果 |
|------|--------|------|
| **human** | 装備マスター | 装備スロット+1（2個装備可能） |

## 5. アビリティの発動条件

### 5.1 条件タイプ

| 条件 | 説明 | 例 |
|------|------|-----|
| HP閾値 | 特定HP以上/以下で発動 | HP30%以下で攻撃+20% |
| 属性条件 | 特定属性技使用時 | 炎技威力+30% |
| 状態条件 | 特定状態時に発動 | 状態異常時に防御+20% |
| 味方条件 | 味方の状態で発動 | 味方登場時HP10%回復 |
| 累積条件 | 回数に応じて強化 | 被ダメージ時攻撃+5%累積 |

### 5.2 発動タイミング

```
【バトル開始時】
- 常時発動型のアビリティが有効化

【ターン開始時】
- 毎T回復系アビリティが発動

【技使用時】
- 属性強化系アビリティが適用

【ダメージ計算時】
- ダメージ増減系アビリティが適用

【被ダメージ時】
- カウンター系、累積系アビリティが発動

【瀕死時】
- 復活系アビリティが発動
```

## 6. 復活アビリティ

### 6.1 復活の仕様

```
【復活可能キャラクター】
三大天ミカ（幻種）のみ

【アビリティ: 女神の祝福】
- 条件: HP70%以上で瀕死ダメージを受けた時
- 効果: HP30%で復活
- 発動: 1バトル1回のみ

【復活後の扱い】
- 瀕死カウントが0に戻る
- 再び瀕死になるとカウント+1
- 「選出済み」のまま（追加選出不可）
```

### 6.2 復活の例

```
1. ミカがHP70%以上の状態で瀕死ダメージを受ける
2. 「女神の祝福」発動 → HP30%で復活
3. ミカ再び瀕死 → 瀕死カウント+1
4. 通常の敗北判定に戻る
```

## 7. アビリティの無効化

### 7.1 アビリティ封じ

```
【グラムの「アビリティ封じ」】
- 効果: 相手のアビリティを4T無効化
- 発動: バトル開始時に自動発動
- 対象: 相手の場に出ているモンスター

【無効化されるアビリティ】
- 常時発動型: 効果が一時停止
- 条件発動型: 条件を満たしても発動しない
- 復活系: 復活が発動しない

【無効化されないアビリティ】
- 種族特性（装備マスター、変幻の魂など）
```

## 8. アビリティのバランス

### 8.1 カテゴリ別強さの目安

| カテゴリ | アビリティの強さ |
|---------|-----------------|
| 幻種 | 最強（固有の特殊効果） |
| 希少種 | 強力（属性+20%以上、複合効果） |
| 一般種 | 標準（属性+15%、単純な効果） |

### 8.2 効果量の目安

| 効果 | 一般種 | 希少種 | 幻種 |
|------|--------|--------|------|
| 属性技強化 | +15% | +20-25% | +30% |
| ステ強化 | +10-15% | +15-20% | +20-25% |
| 状態異常付与率 | +10-15% | +20-25% | - |
| 回復効果 | +10-15% | +25-30% | - |
| 特殊効果 | なし | あり | 固有 |
