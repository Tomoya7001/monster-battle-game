## 1. 概要

アビリティはモンスターが持つパッシブ能力。常時発動または特定条件で自動発動し、バトルを有利に進める。

### 1.1 設計思想（v4.0シンプル化）

| 原則 | 内容 |
|------|------|
| 効果数 | **1～3**（基本は最大2まで、例外あり） |
| HP条件 | **3種類のみ**（満タン/半分以上/半分以下） |
| 倍率表示 | **内部処理**（プレイヤーには見せない） |
| 効果文 | **短く直感的に** |

### 1.2 アビリティ枠とプール

| 項目 | 内容 |
|------|------|
| アビリティ枠 | **1つのみ** |
| プール数 | モンスターごとに2〜5個 |
| 選択方法 | 所持プールから装備のように選択 |

### 1.3 アビリティの習得方法

| 方法 | 説明 |
|------|------|
| ガチャ取得時 | プールからランダムで1つ習得、デフォルトでセット |
| 継承 | 同モンスターを素材にしてアビリティを確定で渡す（最大5体まで選択可） |
| アイテム | 課金アイテム、パス景品、ショップ、イベントなどで好きなアビリティを選択可 |

## 2. HP条件（3種類のみ）

| 条件名 | 内部値 | 説明 |
|--------|--------|------|
| HP満タン時 | hp_full | HP100% |
| HP半分以上 | hp_half_or_more | HP≥50% |
| HP半分以下 | hp_half_or_less | HP≤50% |

## 3. ダメージ倍率（内部処理）

| 表示名 | 内部倍率 | 用途 |
|--------|----------|------|
| 強化 | ×1.3 | 攻撃強化 |
| 大幅強化 | ×1.5 | 条件付き攻撃強化 |
| 軽減 | ×0.8 | 被ダメージ軽減（-20%） |
| 大幅軽減 | ×0.7 | 被ダメージ大幅軽減（-30%） |

## 4. 種族パッシブと種族アビリティの違い

### 4.1 種族パッシブ（v4.0改修 - 効果1つのみ）

種族パッシブは、その種族のモンスター全員が**自動で持っている**特性。選択不要。**アビリティ封じでも無効化されない**。

**設計方針:**
- **効果は必ず1つのみ**
- **ステータス強化・バフ効果は無し**
- **シンプルで覚えやすい**

| 種族 | パッシブ名 | 効果 | 説明文 |
|------|-----------|------|--------|
| human | 装備の達人 | 装備枠+1 | 装備を1つ多く付けられる |
| angel | 天の恵み | 毎ターンHP5%回復 | 毎ターン終了時、HPが少し回復する |
| demon | 略奪本能 | 攻撃命中時バフ1つ奪う | 攻撃を当てると相手の強化を1つ奪う |
| **dragon** | **竜の威厳** | **状態異常は1種類まで** | **同時に2つ以上の状態異常にならない** |
| mechanoid | 機械の体 | 状態異常-1ターン | 状態異常の持続が1ターン短くなる |
| spirit | 霊気 | 状態異常+1ターン | 与える状態異常が1ターン長く続く |
| mutant | 変異体 | 技スロット+1 | 技を1つ多くセットできる |

### 4.2 dragon「竜の威厳」の詳細

```
【竜の威厳】

効果: 状態異常は1種類までしか受けない

仕様:
- 既に状態異常を受けている状態で新たな状態異常を受けると、
  新しい状態異常で「上書き」される
- 2つ目の状態異常が「無効」になるわけではない（上書き）

例:
1. dragonが火傷を受ける → 火傷状態
2. dragonが麻痺を受ける → 火傷が消え、麻痺状態に変化
3. 相手は「どの状態異常を残すか」を考える必要がある

説明文: 「同時に2つ以上の状態異常にならない」
```

### 4.3 種族アビリティ（パッシブとは別効果）

種族アビリティは、その種族のモンスターが**習得できる**アビリティ。アビリティ枠を使用。**全種族でパッシブとは異なる効果**。

| 種族 | アビリティ名 | 効果 |
|------|-------------|------|
| human | 適応力 | 同じ技を2回受けると、その技のダメージ半減 |
| angel | 聖なるオーラ | 回復技の効果が強化される（×1.3） |
| demon | 魂喰らい | 相手を倒した時、全ステータス↑1 |
| dragon | 竜鱗 | 攻撃技が強化される（×1.3） |
| mechanoid | 電磁シールド | 雷属性のダメージを半減 |
| spirit | 霊体 | 攻撃技のダメージを軽減（×0.8） |
| mutant | 変幻の魂 | 被ダメージ時、ランダムで1ステータス↑1 |

## 10. アビリティ無効化

### 10.1 グラムの「アビリティ封じ」

- 効果: 相手のアビリティを4ターン無効化
- 発動: 登場時に自動発動
- 対象: 相手の場に出ているモンスター

### 10.2 無効化されるもの

- 常時発動型: 効果が一時停止
- 条件発動型: 条件を満たしても発動しない
- 復活系/耐え系: 発動しない
- 種族アビリティ: 無効化される

### 10.3 無効化されないもの

- **種族パッシブ**（装備枠+1、技枠+1、HP自動回復、竜の威厳、状態異常短縮/延長など）
