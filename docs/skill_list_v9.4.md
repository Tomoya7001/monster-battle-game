# 技一覧マスターデータ v9.4
最終更新: 2025-12-20

## 変更履歴
- v9.4: 【新規追加】カウンター戦略スキル4種（力量転嫁、魔力反射、撃砕衝、侵蝕の刻印）
- v9.4: 【新規追加】属性物理技7種（紅蓮拳、氷砕拳、雷鳴拳、旋風拳、剛岩拳、聖拳、冥府拳）
- v9.4: 【新規追加】共通技4種（猛襲、連撃、精神統一、活力賦活）
- v9.0: カテゴリシステム移行（★表記廃止→legendary/rare/common）
- v9.0: 復活技削除（復活はミカのアビリティのみ）
- v8.0: 可変コスト技の最大コスト6→5に変更（最大威力S→A）

## 設計思想
- **カテゴリ = 入手難易度**であり、強さの序列ではない
- legendary/rare/common間の性能差は最大5%
- 同コストなら同等の強さ（カテゴリ無関係）
- 奥義威力基準: コスト4→B, コスト5→A, コスト6→S
