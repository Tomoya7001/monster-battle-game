## 1. 基本ルール

### 1.1 バトル形式

| 項目 | 値 |
|------|-----|
| 対戦形式 | 1対1 |
| 勝利条件 | 相手モンスター3体先に倒す |
| パーティ編成 | 5体 |
| 選出方式 | バトル中に最大3体が決まる（事前選出なし） |
| HP | **150固定**（全モンスター共通） |

### 1.2 コストシステム

| 項目 | 値 |
|------|-----|
| 初期コスト | 4 |
| 最大コスト | 7 |
| 毎ターン回復 | 2 |
| 交代 | 行動消費、交代後のコストは4にリセット |

### 1.3 奥義制限

- **バトル中1回のみ使用可能（パーティ全体で1回）**
- どのモンスターが奥義を使ったかに関わらず、2回目以降は使用不可
- 奥義コスト: コスト3～6（モンスターごとに異なる）
- **カテゴリに関係なく、同コストなら同等の強さ**

### 1.4 PvP時のレベル補正

PvPでは全モンスターがLv50として扱われ、公平性を確保する。

| 項目 | 処理 |
|------|------|
| レベル | Lv50として計算 |
| HP | 150固定 |
| 振り分けpt | 200pt扱い |
| 未振り分けpt | **5ステに均等分配** |

## 1-1. ターン処理順序

### ターン開始処理

```
1. コスト回復
   - 全プレイヤー: 現在コスト + 2（最大7）

2. 状態異常ダメージ処理（最大HP基準、端数切り上げ）
   - 毒: 最大HP × 4% = 6ダメージ
   - 火傷: 最大HP × 3% = 5ダメージ
   - 瘴気: 最大HP × 3% = 5ダメージ
   - 悪夢: 最大HP × 3% = 5ダメージ

3. バフ/デバフターン減少
   - 残りターン - 1
   - 0になったら解除

4. フィールド効果処理
   - 設置技発動（棘の罠など）
```

### 行動実行フェイズ

```
1. 行動順決定
   - 交代は優先度+4として扱う（全ての技より先に実行）
   - 技の優先度判定（+3 > +2 > +1 > 0 > -1）
   - 同優先度は素早さで判定
   - 同速はランダム（50%）

2. 交代処理（該当者がいる場合）
   - 交代したプレイヤーはそのターン攻撃できない
   - 相手は交代後のモンスターに攻撃する
   - 交代後のモンスターのコストは4にリセット
   - 交代後のモンスターのバフ/デバフはリセット
   - 状態異常は引き継ぐ

3. 先制行動実行
4. 瀕死判定（復活アビリティ確認）
5. 後攻行動実行
```

### ターン終了処理

```
1. ターン終了時効果
   - 回復系アビリティ（再生細胞など）
   - 種族パッシブ（angel: 最大HP5%回復 = 8HP）

2. 状態異常ターン減少

3. 激昂判定
   - 8T: Lv1発動
   - 12T: Lv2発動
   - 16T: Lv3発動
```

## 1-4. 技の分類

| 分類 | 説明 | 攻撃側 | 防御側 |
|------|------|--------|--------|
| 攻撃技 | 物理ダメージを与える技 | 攻撃 | **防御** |
| 魔力技 | 魔法ダメージを与える技 | 魔力 | **魔防** |
| サポート技 | ダメージを与えない技 | - | - |

**内部値（damage_type）:**
- `physical` = 攻撃技（攻撃 vs 防御）
- `magical` = 魔力技（魔力 vs 魔防）
- `none` = サポート技

### 6.3 拘束の詳細（v8.8改修）

拘束は「**このターンと次のターン交代不可**」になる。

**先制拘束技（影縫い等）の場合:**
```
T1: 影縫い発動 → 拘束付与（残り2T）
T1: 相手は交代できない（このターン）
T2: 相手は交代を選べない（次のターン）
T2終了時: 拘束解除
```

**spirit付与時（3T）:**
```
T1: 拘束付与（残り3T）
T1: 交代不可
T2: 交代不可
T3: 交代不可
T3終了時: 解除
```

種族パッシブは**効果1つのみ**。シンプルで分かりやすい設計。

| 種族 | パッシブ名 | 効果 | 説明文 |
|------|-----------|------|--------|

- 既に状態異常を受けている状態で新たな状態異常を受けると、**新しい状態異常で上書き**される
- 2つ目の状態異常が「無効」になるのではなく「上書き」
- 例: 火傷中に麻痺を受ける → 火傷が消え、麻痺状態に変化

## 11. ダメージ計算式

### 11.1 基本式

**攻撃技（物理）の場合:**
```
基礎ダメージ = (攻撃 / 100) × 威力
最終ダメージ = 基礎ダメージ × バフ補正 × 状態異常補正 
             × タイプ一致 × タイプ相性 × 防御補正 × 乱数 
             × クリティカル × アビリティ補正 × 激昂補正
```

**魔力技（魔法）の場合:**
```
基礎ダメージ = (魔力 / 100) × 威力
最終ダメージ = 基礎ダメージ × バフ補正 × 状態異常補正 
             × タイプ一致 × タイプ相性 × 魔防補正 × 乱数 
             × クリティカル × アビリティ補正 × 激昂補正
```

### 11.2 威力ランク対応表

| ランク | 威力 | コスト目安 |
|--------|------|----------|
| D | 30 | 1 |
| C | 55 | 1-2 |
| B | 80 | 2-3 |
| A | 110 | 3-4 |
| S | 150 | 4-5 |

### 11.3 各補正値

| 要素 | 計算方法 |
|------|----------|
| タイプ一致 | 自属性技×1.5、他属性×1.0 |
| タイプ相性 | **有利×2.0、等倍×1.0、不利×0.5** |
| 防御補正（攻撃技） | 50 / (50 + 防御 × 防御バフ) |
| 魔防補正（魔力技） | 50 / (50 + 魔防 × 魔防バフ) |
| 乱数 | 0.85〜1.00 |
| クリティカル | 発生時×1.5 |

### 11.4 ダメージ計算例

**攻撃特化 vs 耐久特化（等倍、タイプ一致）:**
```
攻撃: 120、威力: 110（ランクA）、防御: 120

基礎ダメージ = (120 / 100) × 110 = 132
防御補正 = 50 / (50 + 120) = 0.294
タイプ一致 = 1.5
乱数平均 = 0.925

ダメージ = 132 × 0.294 × 1.5 × 0.925 ≒ 54

HP150の36% → 約3発で撃破
```

**属性有利追加（×2.0）:**
```
54 × 2.0 = 108ダメージ
HP150の72% → 1-2発で撃破
```

## 12. 実装上の明確化事項（v8.9追記）

### 12.1 HP回復表記の統一

| 表示テキスト | 内部計算 |
|-------------|--------|
| 「HPを少し回復」 | 最大HP × 5% = 8HP |

**適用箇所**: アビリティ効果説明、種族パッシブ説明

### 12.2 コスト0技の仕様

- コスト0技は**アクションを消費する**（ターン終了）
- コストを消費しないだけで、技として扱われる
- 例: ノエル「女神への祈り」（コスト0、防御↑）

### 12.3 「連続使用不可」の定義

**条件**: 以下の**両方**を満たす場合に制限発動
1. 2ターン連続で使用できない
2. **成功時のみ**次ターンの使用が制限される

**例: カウンター**
- T1: カウンター使用 → 相手の攻撃を受けて2倍返し（**成功**）→ T2使用不可
- T1: カウンター使用 → 相手が攻撃技を使わなかった（**失敗**）→ T2使用可能

### 12.4 状態異常効果とバフ段階の区別

状態異常による減少効果（火傷-10%等）は**バフ段階とは別計算**。

```
最終ステータス = 基礎値 × バフ段階倍率 × 状態異常倍率

例: 攻撃100、攻撃↑1（×1.17）、火傷（-10%）
= 100 × 1.17 × 0.9 = 105.3
```

状態異常効果はバフ段階に変換**されない**。

### 12.5 竜の威厳（dragon）の詳細仕様

| 項目 | 仕様 |
|------|------|
| 効果 | 状態異常は同時に1種類まで |
| 上書き | 新しい状態異常が**即時上書き** |
| 同状態付与 | **ターンカウントがリセット**される |

**例:**
- 火傷(残り2T)中に麻痺を受ける → 火傷消去、麻痺(3T)付与
- 毒(残り1T)中に毒を再付与 → 毒(3T)にリセット

### 12.6 交代時のコスト処理

| 項目 | 処理 |
|------|------|
| コスト消費 | **なし** |
| 交代後コスト | **4にリセット** |

交代は行動を消費するが、コストは消費しない。

### 12.7 激昂の適用範囲

激昂ボーナスは以下には**適用されない**：

| 非適用 | 理由 |
|--------|------|
| 状態異常ダメージ（毒等） | 最大HP基準の固定% |
| 回復効果 | 与ダメ/被ダメではない |
| 固定ダメージ（ノエル40等） | 固定値は変動しない |

### 12.8 フィールド効果の重複

| 項目 | 仕様 |
|------|------|
| 最大数 | **各陣地1つまで** |
| 重複時 | **新しい効果が上書き** |

### 12.9 「味方」の定義

| 用語 | 対象 |
|------|------|
| 自分 | 現在場に出ているモンスター本体 |
| 味方 | **控え（ベンチ）のモンスターのみ** |
| 自分を含む | 明記がある場合のみ |

**例:**
- 「味方HP回復」= 控えのHP回復（自分は含まない）
- 「自分と味方」= 自分 + 控え全員

### 12.10 damage_type の定義場所

| 項目 | 内容 |
|------|------|
| 定義場所 | skill_masters JSONファイル |
| 値 | `physical`（攻撃vs防御）、`magical`（魔力vs魔防）、`none`（サポート） |

**補足**: `physical`は「物理攻撃」ではなく「攻撃ステータス参照」の意味。物理的な攻撃イメージではない技でも、攻撃ステを参照するなら`physical`。

### 12.11 個体値「推奨」表記について

| 項目 | 仕様 |
|------|------|
| 目的 | **AI/内部処理用**の参考情報 |
| UI表示 | **表示しない** |

プレイヤーには個体値の育成方針を強制しない設計。

### 12.12 HP回復と復活の区別（重要）

| 概念 | 対象 | 効果 |
|------|------|------|
| **回復** | HP1以上の生存モンスター | HPを増やす |
| **復活** | HP0の瀕死モンスター | HPを1以上にして戦闘可能にする |

**「自分と控え全員HP回復」の対象:**
```
✅ 自分（場に出ている、HP1以上）
✅ 控えA（HP1以上）
✅ 控えB（HP1以上）
❌ 控えC（HP0 = 瀕死）← 回復しない、復活もしない
```

**実装時の注意点:**
```dart
// 回復対象のフィルタリング例
List<Monster> getHealTargets(List<Monster> allies) {
  return allies.where((m) => m.currentHp > 0).toList();
}
```

**現仕様での復活効果:**
- 復活効果は**存在しない**（v9.0で削除済み）
- ミカの「女神の祝福」は「復活」ではなく「耐え」の一種（HP0になる前に発動）
- 将来復活効果を追加する場合は、`effect_type: "revive"`として別実装が必要
