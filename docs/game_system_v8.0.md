## 1. バトル選出ルール

### 1.1 基本ルール

| 要素 | 内容 |
|------|------|
| パーティ編成 | 5体 |
| バトル中に出せる数 | 最大3体 |
| カテゴリ制限 | なし |

### 1.2 選出の流れ

```
例：A, B, C, D, E の5体パーティ

1. Aを出す → 3体選出枠の1体目
2. Aを下げてBを出す → 3体選出枠の2体目
3. Bを下げてCを出す → 3体選出枠の3体目（枠を使い切り）
4. Cを下げてAに戻す → 可能（Aは既に選出済み）
5. Aを下げてBに戻す → 可能（Bも選出済み）
6. D, Eを出す → 不可（3体枠に入っていない）
```

### 1.3 ポイント

- バトル中に「どの3体を使うか」が決まる
- 選出した3体の中ではサイクル（交代→再登場）が可能
- 残り2体は「使わなかった保険」
- 交代時にデバフは解除される（全モンスター共通）

### 1.4 ポケモンとの違い

| 要素 | ポケモン | このゲーム |
|------|---------|-----------|
| 選出タイミング | バトル前に3体決定 | バトル中に決まる |
| 控え | 見せ合いで予測可能 | 5体全て可能性あり |
| 戦略性 | 選出段階で読み合い | バトル中の判断が重要 |

## 2. レベルシステム

| 要素 | 内容 |
|------|------|
| 最大レベル | 50 |
| レベルアップ方法 | 経験値を獲得して上昇 |
| レベルアップ報酬 | 1Lvごとにポイント獲得（振り分け用） |

### バトル時のレベル

- バトルではレベル50固定でステータス計算
- レベルが50未満でもバトル時は50として扱う（対戦の公平性）

## 3. ステータスシステム

### 3.1 基本構造

| 要素 | 内容 |
|------|------|
| ステータス種類 | HP、攻撃、防御、魔力、速さ（5種） |
| 基礎値 | モンスターごとに固定 |
| 個体値補正 | パターンにより1ステ+10%、1ステ-10% |
| ポイント振り分け | レベルアップで獲得したポイントを配分 |
| 上限 | モンスター・ステータスごとに設定 |

### 3.2 最終ステータス計算式

```
最終値 = 基礎値 + (基礎値 × 個体値補正) + ポイント振り分け

※上限を超えない
```

### 3.3 ポイント振り分け

| 要素 | 内容 |
|------|------|
| 獲得量 | 1Lvごとに5ポイント |
| Lv50時の合計 | 250ポイント |
| 振り分け | 各ステータスに自由配分 |
| 上限 | モンスターごとに各ステータスの上限あり |
| 振り直し | **アイテム消費で可能** |

### 3.4 ステータス上限の例

**幻種 テュール（物理アタッカー）**

| ステータス | 基礎値 | 上限 | 振り分け可能幅 |
|-----------|--------|------|---------------|
| HP | 95 | 150 | +55 |
| 攻撃 | 105 | 160 | +55 |
| 防御 | 70 | 110 | +40 |
| 魔力 | 60 | 80 | +20 |
| 速さ | 60 | 100 | +40 |

**一般種 クロム（後攻耐久型）**

| ステータス | 基礎値 | 上限 | 振り分け可能幅 |
|-----------|--------|------|---------------|
| HP | 90 | 160 | +70 |
| 攻撃 | 80 | 130 | +50 |
| 防御 | 85 | 150 | +65 |
| 魔力 | 40 | 60 | +20 |
| 速さ | 40 | 60 | +20 |

**設計意図:**
- テュール：攻撃が最も伸びる、魔力は伸びない
- クロム：HP・防御が大きく伸びる、速さは伸びない（後攻向け）
- 一般種でも特定ステータスで幻種を上回れる

## 4. 個体値パターン（性格）

### 4.1 基本仕様

| 要素 | 内容 |
|------|------|
| パターン数 | モンスターごとに3〜5種類 |
| 効果 | 1ステータス+10%、1ステータス-10% |
| 設計方針 | **ハズレなし**（そのモンスターに合うパターンのみ） |
| 決定タイミング | ガチャ/入手時にランダム決定 |
| 変更 | **不可**（厳選要素） |

### 4.2 設計ルール

```
1. 主力ステータスは下降しない
   例：魔法アタッカーの「魔力↓」は存在しない

2. 使わないステータスを下降させる
   例：魔法アタッカーの「攻撃↓」は問題ない

3. 下降がメリットになるパターンも存在
   例：後攻型の「速さ↓」は実質メリット

4. 全パターンに「使える型」がある
```

### 4.3 パターン例

**幻種 タマモ（魔法アタッカー）**

| パターン名 | 上昇(+10%) | 下降(-10%) | 想定型 |
|-----------|-----------|-----------|--------|
| 妖艶 | 魔力 | 攻撃 | 魔法特化 |
| 疾風 | 速さ | 攻撃 | 先制型 |
| 狡猾 | 速さ | HP | 速攻型 |
| 妖狐 | 魔力 | 防御 | 火力特化 |

**幻種 テュール（物理アタッカー）**

| パターン名 | 上昇(+10%) | 下降(-10%) | 想定型 |
|-----------|-----------|-----------|--------|
| 猛攻 | 攻撃 | 魔力 | 物理特化 |
| 豪傑 | 攻撃 | 速さ | 鈍足火力 |
| 疾駆 | 速さ | 魔力 | 先制型 |
| 堅牢 | HP | 魔力 | 耐久型 |

**一般種 クロム（後攻耐久型）**

| パターン名 | 上昇(+10%) | 下降(-10%) | 想定型 |
|-----------|-----------|-----------|--------|
| 重装 | 防御 | 速さ | 後攻耐久 |
| 不動 | HP | 速さ | HP耐久 |
| 反撃 | 攻撃 | 速さ | 後攻火力 |

**クロムの設計意図:**
- 全パターンが「速さ↓」
- 後攻アビリティと相性が良い
- 「速さ↓」がメリットになる設計

## 5. アビリティシステム

### 5.1 基本仕様

| 要素 | 内容 |
|------|------|
| 所持枠 | 3〜4つまで（解放した分だけ使用可能） |
| 選択 | バトル前に1つ選ぶ（付け替え自由） |
| 初期所持 | 入手時にプールからランダムで1つ |
| 設計方針 | **1アビリティ = 1効果**（シンプル） |

### 5.2 解放方法

| 方法 | 内容 | 結果 |
|------|------|------|
| 継承 | 同モンスターを素材にする | **素材のアビリティを確定で獲得** |
| アイテム | アビリティ解放券を使用 | **プールからランダムで獲得** |

### 5.3 初期アビリティの排出設定

| 種類 | 排出率 | 説明 |
|------|--------|------|
| コモン | 70% | 使いやすい、無条件発動 |
| レア | 30% | 強力だが条件付き |

**設計意図:**
- 初期は「使いやすい」アビリティが出やすい
- 「強力なアビリティ」は継承 or アイテムで狙う
- 厳選要素として機能する

### 5.4 アビリティの質（カテゴリ別）

| カテゴリ | アビリティの特徴 |
|---------|----------------|
| 幻種 | 条件付きで強力（状態異常相手に+20%など） |
| 希少種 | 無条件で安定（先制時+15%など） |
| 一般種 | シンプルで使いやすい |

## 6. 技プールと奥義

### 6.1 技プール

| カテゴリ | 属性数 | 特徴 |
|---------|--------|------|
| 幻種 | 2〜3属性 | 多彩な選択肢 |

**例外:**
- スライムなど一部モンスターは全属性技を所持
- モンスターの個性として機能

### 6.2 技の習得

| 要素 | 内容 |
|------|------|
| 習得方法 | レベルアップで自動習得 |
| 最大習得レベル | Lv50（全技Lv50以内に収める） |
| 技スロット | 基本4つ（ミュータント種は5つ） |

### 6.3 奥義

#### 設計思想

```
奥義の強さはコストで決まる（カテゴリではない）
- 同じコストなら、どのカテゴリでも同等の強さ
- 全カテゴリでコスト3〜6の奥義が均等に存在
- モンスターの個性は「追加効果の種類」で表現
```

#### コスト別基準

| コスト | 威力 | 追加効果 | 初手使用 |
|--------|------|---------|---------|
| 3 | B | なし or 軽い | ✅ 可能 |
| 4 | A | 控えめ | ✅ 可能 |
| 5 | S | あり | ❌ 2T目以降 |
| 6 | SS | 強力 | ❌ 3T目以降 |

#### コスト分布（カテゴリ別）

| カテゴリ | コスト3 | コスト4 | コスト5 | コスト6 |
|---------|--------|--------|--------|--------|
| 幻種(7体) | 1 | 2 | 2 | 2 |
| 希少種(15体) | 2 | 5 | 6 | 2 |

## 7. 状態異常システム

### 7.1 基本ルール

| 項目 | ルール |
|------|--------|
| 付与確率 | 100%確定（運要素排除） |
| 重ねがけ | 同じ状態異常の重ねがけ不可（持続リセットのみ） |
| 同時付与上限 | 2種類まで |
| 3種類目付与時 | 最も古い状態異常が解除される |

### 7.2 状態異常一覧

| 状態異常 | 効果 | 持続 |

| 毒(poison) | 毎T HP2%ダメージ | 3T |

| 瘴気(miasma) | 毎T HP3%ダメージ + 魔力-5% | 3T |
| 麻痺(paralysis) | 素早さ-15% | 3T |
| 拘束(restraint) | 交代不可 | 2T |
| 封印(seal) | 最高コスト技使用不可 | 2T |
| 魅了(charm) | 被ダメージ+10% | 2T |
| 恐怖(fear) | 与ダメージ-10% | 2T |
| 悪夢(nightmare) | 毎T魔力3%ダメージ + 回復効果-50% | 3T |

### 7.3 状態異常の分類

| 分類 | 状態異常 | 特徴 |

## 9. カテゴリ（入手難易度）

### 9.1 カテゴリ一覧（v8.4）

| カテゴリ | レアリティ | モンスター数 | 排出率 | 1体あたり |
|---------|-----------|-------------|--------|----------|
| 一般種 | ★2-3 | 22体 | 65% | 2.95% |
| 希少種 | ★4 | 10体 | 27% | 2.70% |
| 幻種 | ★5 | 7体 | 8% | 1.14% |
| **合計** | - | **39体** | 100% | - |

### 9.2 レアリティ別モンスター数

| レアリティ | カテゴリ | 体数 |
|-----------|---------|------|
| ★2 | 一般種 | 7体 |
| ★3 | 一般種 | 15体 |
| ★4 | 希少種 | 10体 |
| ★5 | 幻種 | 7体 |

### 9.3 設計思想

```
カテゴリ = 入手難易度であり、強さの序列ではない

- 「幻種だから強い」ではなく「幻種だから珍しい」
- 性能差は最大5%（覆せる範囲）
- 一般種でも十分に戦える
- 幻種の価値は「コレクション・演出・所有欲」
```

### 9.4 なぜ一般種でも戦えるか

```
1. 性能差は最大5%（ポイント振り分けで覆せる）
2. ステ上限の一部が幻種より高い一般種が存在
3. アビリティが使いやすい（無条件発動）
4. 相性で幻種に勝てる設計
5. 個体値パターンでメリットを得られる
6. 奥義の強さはコストで決まる（カテゴリ無関係）
7. 一般種にもコスト6の強力奥義が存在
```

### 9.5 パーティ制限

| 制限 | 内容 |
|------|------|
| 幻種 | 制限なし |
| 希少種 | 制限なし |
| 一般種 | 制限なし |

## 10. 種族と種族特性

### 10.1 種族一覧

| 種族 | 特性（パッシブ） | 説明 |
|------|-----------------|------|
| 人間 | 装備枠+1 | 基本1枠→2枠装備可能 |
| 天使 | 回復量+15 | 回復技の回復量が+15される |
| 悪魔 | デバフ+1T | 付与したデバフが1ターン長い |
| 竜 | 被クリティカル無効 | クリティカルを受けない |
| メカノイド | 被状態異常-1T | 受ける状態異常の持続が1T短い |
| スピリット | 状態異常+1T | 付与した状態異常が1ターン長い |
| ミュータント | 技枠+1 | 基本4枠→5つの技を持てる |

### 10.2 種族の役割

| 種族 | 得意 | 苦手 |
|------|------|------|
| 人間 | 装備で柔軟に対応 | 素のスペックは普通 |
| 天使 | 回復・耐久 | 火力 |
| 悪魔 | デバフ・弱体化 | 正面火力 |
| 竜 | 火力・耐久 | 素早さ |
| メカノイド | コントロール | 状態異常耐性は高いが特化ではない |
| スピリット | 魔法火力・状態異常 | 耐久 |
| ミュータント | 万能・適応 | 突出した強みがない |

### 10.3 種族の詳細説明

#### 人間
```
コンセプト：装備で何にでもなれる

特性（パッシブ）：装備枠+1（基本1枠→2枠）
強み：装備次第でアタッカーにもタンクにもなれる
弱み：装備がないと凡庸

プレイスタイル：装備厳選が重要
```

#### 天使
```
コンセプト：味方を守り、癒す

特性（パッシブ）：回復量+15
強み：回復量が多い、耐久力がある
弱み：攻撃技の威力が低め

プレイスタイル：サポート・耐久戦
```

#### 悪魔
```
コンセプト：デバフで相手を弱体化

特性（パッシブ）：デバフ+1T
強み：攻撃↓、防御↓などを長く維持
弱み：デバフが入らないと火力が出ない

プレイスタイル：搦め手、持久戦
```

#### 竜
```
コンセプト：圧倒的なパワーとタフネス

特性（パッシブ）：被クリティカル無効
強み：急所を突かれない安定感、殴り合いに強い
弱み：先手を取られやすい

プレイスタイル：鈍足高火力、後攻で耐えて返す
```

```
コンセプト：相手を管理・制御する

強み：状態異常を受けても影響が短い
弱み：特化した強みがない

プレイスタイル：デバフで相手を弱らせてから倒す
```

プレイスタイル：速攻、状態異常で削る
```

プレイスタイル：相手に合わせて戦い方を変える
```

## 12. コストシステム

### 12.1 基本仕様

| 要素 | 値 |
|------|-----|
| 初期コスト | 4 |
| 最大コスト | 7 |
| 毎ターン回復 | +2 |
| 交代コスト | 4（コストが4にリセット） |

### 12.2 技のコスト帯

| コスト | 特徴 |
|--------|------|
| 1〜2 | 低威力、初手から連打可能 |
| 3〜4 | 中威力、メイン技 |
| 5〜6 | 高威力、奥義クラス |

### 12.3 コスト推移例

```
ターン1: コスト4 → 3コスト技使用 → 残り1
ターン2: 1+2=3 → 2コスト技使用 → 残り1
ターン3: 1+2=3 → 3コスト技使用 → 残り0
ターン4: 0+2=2 → 2コスト技使用 → 残り0
ターン5: 0+2=2 → コスト溜め（攻撃せず） → 残り2
ターン6: 2+2=4 → 交代 → 次のモンスターはコスト4
```
