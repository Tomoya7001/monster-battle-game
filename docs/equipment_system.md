# 装備・アイテムシステム仕様書

**作成日**: 2025-12-16  
**バージョン**: v1.0  
**ステータス**: Phase1 確定版

---

## 1. 概要

装備・アイテムシステムは、モンスターの強化とバトル戦略の幅を広げるシステム。装備でステータス強化、アイテムで回復・サポートを行う。

---

## 2. 装備システム

### 2.1 装備の基本

| 項目 | 内容 |
|------|------|
| 装備スロット | 通常1個、human種族は2個 |
| 装備カテゴリ | 武器 / 防具 / アクセサリー / 特殊 |
| 入手方法 | 錬成、ボスドロップ、イベント報酬 |

### 2.2 装備データ構造

```json
{
  "equipment_id": "eq_001",
  "name": "炎の剣",
  "category": "weapon",
  "rarity": 3,
  "effects": [
    {
      "effect_type": "stat_boost",
      "stat": "attack",
      "value": 15,
      "value_type": "percentage"
    }
  ],
  "restrictions": {
    "species": null,
    "element": "fire"
  },
  "craft_materials": [...],
  "craft_cost": 15000
}
```

## 6. 錬成システム

### 6.1 錬成の流れ

```
1. 錬成画面で装備を選択
2. 必要素材を確認
3. ゴールドを消費して錬成
4. 装備獲得
```

### 6.2 錬成コスト目安

| レア度 | 素材数 | ゴールド |
|--------|--------|---------|
| ★2 | 5-10個 | 3,000-5,000 |
| ★3 | 10-20個 | 10,000-20,000 |
| ★4 | 20-30個 | 40,000-60,000 |
| ★5 | 30-50個 | 100,000-150,000 |

### 6.3 必要素材の種類

```
【汎用素材】
- 鉄鉱石（★1-3）
- 魔法の鉱石（★3-4）
- 秘宝の鉱石（★4-5）

【属性素材】
- 炎の欠片、水の欠片、雷の欠片...
- 各属性装備の錬成に必要

【種族素材】
- ドラゴンの牙、天使の羽根...
- 専用装備の錬成に必要

【ボス素材】
- ボスの証
- ★4-5装備に必要
```

## 7. 装備強化（進化）

### 7.1 進化の仕組み

```
【段階的進化】
★2装備 + 素材 → ★3装備
★3装備 + 素材 → ★4装備
★4装備 + 素材 → ★5装備

【進化時】
- 元の装備は消費される
- 効果量がUP
- 名前が変化する場合あり
```

### 7.2 進化コスト

| 進化 | 素材 | ゴールド |
|------|------|---------|
| ★2→★3 | 10個 + 中レア5個 | 15,000 |
| ★3→★4 | 20個 + 高レア5個 | 50,000 |
| ★4→★5 | 30個 + ボス素材3個 | 150,000 |

## 8. 素材システム

### 8.1 素材の入手方法

| 入手方法 | 素材の質 | 効率 |
|----------|---------|------|
| ステージクリア | 汎用素材 | 高 |
| モンスター撃破 | 属性/種族素材 | 中 |
| ボス撃破 | ボス素材 | 低 |
| 探索（派遣） | 汎用素材 | 中 |
| ショップ購入 | 汎用素材のみ | 低 |

### 8.2 ドロップ率

| 素材レア度 | ドロップ率 |
|-----------|-----------|
| 低レア | 50-80% |
| 中レア | 30-50% |
| 高レア | 10-30% |
| ボス素材 | 5-20% |

### 8.3 天井システム

```
【同じ素材を狙う場合】
- 10回目: ドロップ率×2
- 20回目: ドロップ率×3
- 30回目: 確定ドロップ
```
